Global Azure Austria 2021

Friday, April 16, 2021

Find out more

Azure and Cloud Computing Conference

Austrian Edition of Global Azure 2021

Global Azure is a 3-day virtual conference in which communities around the world organize live streams to learn about Cloud Computing and Microsoft Azure. This is the Austrian edition of Global Azure 2021 provided by the Coding Club Linz.

We Are Virtual!

The current Covid-situation does not allow us to invite you all to Linz (Austria). Like last year, Global Azure 2021 will be a virtual event again. You can watch all sessions on YouTube, ask questions in comments, and join live discussions on Discord.

Everything Live!

At Global Azure Austria, all sessions including live demos will be performed live. Therefore, speakers can answer questions and interact with the audience during their talks.

German and English Content

The majority of our sessions will be in German. However, in order to offer content for a broad audience, some speakers have agreed to deliver their sessions in English so that people who do not understand German fluently can participate in the event, too.

Join us on Friday, April 16, 2021!

Speakers

Internationally highly respected speakers will share their knowledge about topics around Azure and Cloud Computing.

Sponsors

William Assaf

Senior Content Developer, Microsoft
William Assaf is a senior content developer on the SQL-Docs team at Microsoft. Formerly a Data Platform MVP and a consultant, and before that a .NET developer, William currently works with Microsoft product groups on content related to Microsoft SQL offerings, specifically over SQL performance, and Azure SQL DB, Big Data Clusters, and Azure Synapse. As a consultant for 13 years he worked with clients on SQL Server and Azure SQL platform optimization, management, disaster recovery, and high availability, and led a multi-city team of senior consulting SQL DBA’s. William has written for Microsoft SQL certification exams since 2011 and was the team lead author of the 2017 and 2019 editions of “SQL Server Administration Inside Out” by Microsoft Press. William is a past lead of the annual SQLSaturday Baton Rouge Planning Committee and is a big fan of data platform communities far and wide.
