RESTful Web Services mit Spring Boot

Friday, April 16, 2021 16:00 - 16:45 CEST

Die Kommunikation zwischen Front- und Backend einer App erfolgt oft über eine sogenannte REST API. Dabei stellt das Backend bestimmte Methoden zur Verfügung, die Clients benutzen können. Doch wie funktioniert so eine REST API eigentlich und was sollte man dabei beachten?

In diesem Workshop werden wir einen einfachen Web Service mit Spring Boot implementieren.

Um an diesem Workshop teilzunehmen, solltest du folgendes auf deinem Computer installieren: * Java mit Version 1.8 oder höher * Eine Enwicklungsumgebung wie Intellij IDEA (bevorzugt) oder Eclipse (wenn du schon Erfahrung damit hast)
