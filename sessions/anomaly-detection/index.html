Anomaly Detection as a Service - Was ist normal?

Friday, April 16, 2021 15:00 - 15:50 CEST

Wann wurde etwas aus Ihrem Shop gekauft? Wann war ein Zugriff auf Ihrer Website? Wann wurde eine Nachricht versandt? Wann ist ein Fehler aufgetreten? Wir speichern täglich derartige zeitbasierte Daten in unseren Anwendungen - aber wie schnell würde es uns auffallen, wenn es zu ungewöhnlichen Abweichungen kommen würde?

Der Metrics Advisor aus den Azure Cognitive Services stellt APIs zur AI-unterstützten Erkennung von Abweichungen in Daten bereit. Roman Schacherl zeigt live und an unterschiedlichen Praxisbeispielen, wie Sie diese neuen Azure-Dienste nutzen können.
